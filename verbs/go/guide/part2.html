2) TOEIC頻出（職場・メール・会議で刺さる）
go ahead with（〜を進める）
コアイメージ：「〜と共に前方へ」進む

計画 ━━━→ 前方 ahead
    （with = 共に）


We'll go ahead with the project.
プロジェクトを進めます。
→ プロジェクトと共に前方へ進みます


Let's go ahead with the plan.
計画を進めましょう。
→ 計画と共に前方へ進みましょう


The company decided to go ahead with the merger.
会社は合併を進めることを決定した。
→ 会社が合併と共に前方へ進むことを決めた


TOEIC頻出：
プロジェクト推進・意思決定の文脈
ポイント：
proceed with とほぼ同義
「実行に移す」ニュアンス

go into effect（発効する、施行される）
コアイメージ：「効力の内部へ」入っていく

無効 ━━→ [効力の内部]
    （into effect = 効力へ）


The new policy goes into effect next month.
新しい方針は来月発効します。
→ 新方針が来月効力の内部へ進む


When will the changes go into effect?
変更はいつ発効しますか？
→ 変更がいつ効力の内部へ進むか


The law went into effect last year.
その法律は昨年施行された。
→ 法律が昨年効力の内部へ進んだ


TOEIC頻出：
規則施行・方針変更の文脈
ポイント：

「正式に有効になる」

コアイメージ：「間違った方向へ」離れていく

正常 ━━━━→ 異常
       （wrong = 間違い）

→ 計画が間違った方向へ進んだ

→ 何かが間違った方向へ進んだら、電話して

4) 重要だけど”意味がブレやすい”注意組
go off（多義語の整理）
コアイメージ：正常状態から「離れる（off）」
意味1：爆発する

The bomb went off.
→ 静止状態から離れて爆発


意味2：アラームが鳴る

My alarm went off.
→ 静止状態から離れて作動


意味3：腐る

The milk went off.
→ 新鮮な状態から離れて腐敗


意味4：立ち去る

He went off angry.
→ その場から離れて去る


意味5：停電する

The lights went off.
→ 点灯状態から離れて消灯


見分け方：
	∙	爆弾・銃 = 爆発する
	∙	アラーム・ベル = 鳴る
	∙	食べ物 = 腐る
	∙	人 = 立ち去る
	∙	電気・機械 = 消える／停止する

go on（多義語の整理）
コアイメージ：「前方へ進み続ける（on）」
意味1：続ける

Go on with your work.
→ 仕事を前方へ進め続けて


意味2：起こる

What's going on?
→ 何が進行しているのか


意味3：点灯する

The lights went on.
→ 明かりが点いた（作動状態へ進んだ）


意味4：さらに話を続ける

Go on, tell me more.
→ 続けて、もっと話して


見分け方：
	∙	作業・活動 = 続ける
	∙	疑問文 = 起こる
	∙	電気・機械 = 点く／作動する
	∙	会話 = 話を続ける

go through（多義語の整理）
コアイメージ：「貫通して（through）」進む
意味1：通り抜ける

Go through the door.
→ ドアを貫通して進む


意味2：経験する

She went through a lot.
→ 多くの困難を通り抜けた


意味3：調べる

Go through the documents.
→ 書類を貫通して進む（隅々まで見る）


意味4：承認される

The proposal went through.
→ 提案が承認プロセスを貫通した


意味5：使い果たす

I went through all my money.
→ お金を全部使い果たした（通過した）


見分け方：
	∙	物理的な場所 = 通り抜ける
	∙	困難・経験 = 経験する
	∙	書類・情報 = 調べる
	∙	提案・計画 = 承認される
	∙	お金・資源 = 使い果たす

go out（多義語の整理）
コアイメージ：内部から「外へ（out）」離れる
意味1：外出する

Let's go out.
→ 外へ離れよう


意味2：火が消える

The fire went out.
→ 燃焼状態から離れた


意味3：交際する

They're going out.
→ 彼らは付き合っている


意味4：流行遅れになる

That style went out.
→ そのスタイルは流行から離れた


意味5：引き潮になる

The tide is going out.
→ 潮が引いている（海岸から離れている）


見分け方：
	∙	人 + 場所なし = 外出する
	∙	火・明かり = 消える
	∙	人 + with = 交際する
	∙	スタイル・流行 = 流行遅れになる
	∙	潮 = 引く

go over（多義語の整理）
コアイメージ：「上を越えて（over）」進む
意味1：復習する

→ レッスンの上を越えて再度進む


意味2：調べる

意味3：予算を超える

意味4：好評を得る

→ スピーチが好評に越えて進んだ


見分け方：
	∙	教材・資料 = 復習する／調べる
	∙	予算・限度 = 超える
	∙	well/badly = 評判が良い／悪い
